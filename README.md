# Smallbiz-infosec www.secure-metrics.com
Started (fall 2024) in order to gain hands-on understanding of the AWS platform, this AWS project is now complete (fall 2025). The AWS environment and hosted website have been permanently shut down, and all related resources and credentials have been removed (free tier had ended).

## Portfolio Project
A simple static website to demonstrate AWS knowledge.

## AWS Services used and configured
- S3 Bucket
- Cloudfront
- Amplify
- WAF (through Amplify)
- Cloudwatch
- Route 53 (for domain secure-metrics.com)
- Billing and Cost Management
- AWS CLI

## Additional Services used
- Gitlab
- Z Shell
